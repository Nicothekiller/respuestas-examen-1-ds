# Corrección examen parcial 1

En este repositorio se puede encontrar la corrección del primer examen parcial.
Las respuestas se pueden encontrar en el archivo "main.pdf".
Las respuestas de la sección practica también se pueden encontrar en el pdf,
al igual que en sus archivos respectivos.

El pdf fue realizado con la herramienta [typst](https://typst.app/home/),
y todo el código utilizado para generar el documento puede ser encontrado en el repositorio.
El archivo "main.typ" contiene las partes principales del documento, mientras que
el archivo "template.typ" contiene un template estilo apa creado por el autor del repositorio.
